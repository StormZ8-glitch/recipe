# Personal Money Manager - Frontend

This is the frontend part of the Personal Money Management App designed for students and young professionals in Cameroon. The application allows users to track their income and expenses, categorize transactions, set savings goals, and view dashboards and summaries.

## Features

- **Manual Income and Expense Tracking**: Users can manually input their income and expenses.
- **Categorization**: Transactions can be categorized for better tracking and analysis.
- **Dashboards**: Visual representation of financial data, including total income, expenses, and balance.
- **Summaries**: Quick summaries of financial activity over a selected period.
- **Savings Goals**: Users can set and track their savings goals.
- **Future Integration with MoMo APIs**: Planned integration for seamless transaction imports.

## Getting Started

### Prerequisites

- Node.js and npm installed on your machine.
- A running instance of the backend API.

### Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd personal-money-manager-cm/frontend
   ```

2. Install dependencies:
   ```
   npm install
   ```

### Running the Application

To start the development server, run:
```
npm start
```

The application will be available at `http://localhost:3000`.

### Folder Structure

- **public/**: Contains the static files, including `index.html`.
- **src/**: Contains the source code for the React application.
  - **components/**: Reusable components used throughout the application.
  - **pages/**: Different pages of the application.
  - **services/**: API service functions for making requests to the backend.
  - **hooks/**: Custom hooks for managing state and side effects.
  - **styles/**: Global styles for the application.
  - **tests/**: Unit tests for the components.
